### 1. Apa itu Node.js?
Node.js itu alat buat jalanin **JavaScript di luar browser**.  
Biasanya dipake buat bikin **backend**, **API**, dan **server-side logic**.

---

### 2. Bedanya Node.js sama JavaScript di Browser?
- **Browser JS** → ngurus tampilan (UI), DOM, klik-klik, animasi.  
- **Node.js** → ngurus backend, database, file system, server.  
Bahasanya sama, tapi **lingkungannya beda**.

---

### 3. Fungsi `require()` di Node.js?
`require()` itu buat **import module atau file lain** ke dalam file yang lagi lu kerjain.  
Ibarat “bro sini bantu gue”, dan modulnya langsung ikut masuk.
