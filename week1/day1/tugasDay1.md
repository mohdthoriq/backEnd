Gw jadi tahu bahwa backend itu sebenernya bagian penting dari sebuah aplikasi yang kerja di belakang layar. Kalau frontend itu bagian yang kita lihat, kayak tombol, warna, tulisan, dan tampilan aplikasi, backend itu bagian yang ngurus semua proses yang gak kelihatan. Backend lah yang ngatur data, ngecek input user, ngirim jawaban ke frontend, dan ngatur aturan-aturan aplikasi. Intinya, backend itu otak yang nentuin gimana aplikasi berjalan dengan benar.

Waktu gw mulai belajar, gw kira backend cuma soal database. Tapi ternyata backend lebih dari itu. Backend itu yang nerima permintaan dari frontend, terus backend mikirin harus ngapain: mau nyimpen data, ngupdate data, ngapus data, atau ngambil data. Setelah backend selesai prosesnya, backend balikin hasilnya lagi ke frontend. Itu sebabnya aplikasi bisa jalan dengan mulus. Tanpa backend, aplikasi cuma jadi tampilan aja tanpa fungsi beneran.

Hubungan antara frontend sama backend itu kayak dua tim yang harus kompak. Frontend itu yang ngomong ke user, kayak, “Eh lu mau daftar? Mau login? Mau liat produk?” Nah setelah user klik sesuatu, frontend bakal ngomong ke backend, “Bro, ini ada data baru nih. Tolong proses ya.” Backend langsung kerja, misalnya ngecek password bener atau nggak, terus backend balikin hasilnya lagi. Frontend tinggal nampilin hasilnya ke user dalam bentuk tampilan yang rapi.

Jadi bisa dibilang frontend itu wajah dan suara aplikasi, sementara backend itu pikiran dan tenaganya. Frontend bikin user nyaman, backend bikin semuanya jalan sesuai aturan. Kalau frontend gak ada backend, user cuma bakal lihat tampilan kosong tanpa data. Sebaliknya, kalau backend gak ada frontend, semua data dan proses gak bakal ada yang nampilin ke user. Makanya dua-duanya harus saling melengkapi.

Belajar backend itu penting buat pemula karena kita jadi ngerti cara aplikasi beneran bekerja, bukan cuma tampilannya. Kita jadi paham gimana data bisa muncul, gimana login bisa aman, gimana pesanan bisa diproses. Dari situ, bikin aplikasi jadi jauh lebih masuk akal dan lebih lengkap. Semakin gw belajar, semakin gw tahu bahwa backend itu fondasi yang bikin aplikasi beneran hidup. Dan kalau frontend dan backend udah saling ngerti perannya masing-masing, aplikasi bisa jadi kuat, berguna, dan gampang dikembangin ke depannya.
